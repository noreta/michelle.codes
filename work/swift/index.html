I completed the Codepath iOS for Designers bootcamp in 2015 and again in 2016. The bootcamp was created to teach designers how to build fully interactive iOS prototypes in Xcode using Swift. After re-creating a number of existing apps, we worked in teams to create our own app. During both sessions, I worked in a team of three to build the final project.

All gifs on this page are screencasts of the Xcode simulator, built in Swift.

Codepath 2016 project: Overlay

Overlay is an app that leads people through tutorials of design fundamentals. The target audience was non-designers like engineers and project managers, who benefit from having a basic understanding of design principles.

Each section had interactive tutorials, with a number of different gestures and animations used. We set up basic logic to trigger correct or incorrect buttons, allowing the user to progress only when they got the answer right. This progress was stored in a custom data model we created:

Codepath 2015 Project: Minimeet

Minimeet is an easy was to create small-group, coversation-based meetups about any topic you want. Users can create a meetup via the app, select a location using a Foursquare integration, and publish it to the app's main feed. It is added to a running feed of events and stored with Parse, and up to five people can RSVP.
