Seven Ounces: A Nutella Project

Product design, visual design, branding, illustration, front end development, copywriting, photography

View Site
View on Github

Seven Ounces is a web app that helps users find new and exciting ways to add Nutella into their everyday lives. Users can search nearby locations that serve Nutella or find a great recipe to make at home. I participated in all aspects of the site development and design, with a focus on visual design, user experience, and frontend development. The styling is implemented with Twitter Bootstrap, SCSS, and CSS3, and the app is built with Ruby on Rails and jQuery, and deployed to Heroku.

This project was a collaboration with Marie.

Sketching and product design

After coming up with the basic concept and problem statement, we sketched out various ways to implement a solution and create a user-friendly experience.

Design Iterations in the Browser

After coming up with a basic site architecture and branding, we set up the site and iterated on the design in the browser.

The landing page gives users two options - searching for restaurants or recipes - and subsequent pages display images and content on a responsive grid.

Food Photography

All photography on the site was was styled and shot by us. We used the photography as background images and as a part of the site footer.

Implementation: APIs, jQuery, and Rails

The front-end was implemented with Twitter Bootstrap and SCSS. The back-end is Ruby on Rails, and the app is deployed to Heroku. We also used APIs from Yelp and Yummly to power search results.
